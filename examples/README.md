# Examples

`tasgi` ships with multiple example apps so you can choose the style you want to study first.

## `service_api`

Advanced example with:

- auth
- router modules
- streaming
- realtime WebSocket notifications
- thread-executed business routes
- OpenAPI and Swagger UI

Run:

```bash
python3 examples/service_api/main.py
```

## `modular_api`

Cleaner composition example with:

- app factory
- routers in separate modules
- services in separate modules
- route registration outside the app module

Run:

```bash
python3 examples/modular_api/main.py
```
